# robot_rts

## Control codes

1 - runForward

2 - runBack

3 - turnRight

4 - turnLeft

5 - runStop


## Error codes

  | vbFL | vbFR | vbBL | vbBR | virFront | virBack | = |   x  |   x  |   x  |   x  |    x     |    x    |   
  
  vbFL =     100000 = 32
  
  vbFR =     010000 = 16
  
  vbBL =     001000 = 8
  
  vbBR =     000100 = 4
  
  virFront = 000010 = 2
  
  virBack =  000001 = 1
